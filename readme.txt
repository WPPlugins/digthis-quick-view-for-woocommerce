== Description ==
Adds quick view functionality to Shop and WooCommerce loop.
Uses magnific popup and is specially good for mobile devices.
Compatible with WooCommerce 3.0.1

1. Just install the plugin and a quick view button will be added to WooCommerce Shop Loop


== Installation ==

1. Upload `digthis-woocommerce-quick-view` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You're done
